Project updates

Building the boring parts until they become the product.

A public record of reliability, distribution, and interface work.

CurrentReliability foundation

Agent jobs became recoverable units of work.

Added job attempts, sixty-second leases, five-second heartbeats, running cancellation, stale recovery, and idempotent terminal writes. A vanished worker can no longer strand a job forever or overwrite a newer attempt.

serverworkerdatabase
CurrentEvidence layer

Receipts stopped trusting agent claims.

Changed files now come from Git, verification comes from configured worker commands, and status comes from observed exit state. Agent prose remains useful context, not system truth.

receiptsverification
CurrentSource isolation

Every job moved into a disposable worktree.

Jobs start from source HEAD, preserve dirty and untracked local files, capture relative binary patches, support committed subdirectories, and remove the temporary worktree after execution.

gitsafety
CurrentOpen-source distribution

The repository gained a real operator path.

Added durable Compose/PostgreSQL deployment, automatic Alembic migrations, a worker setup command, service guidance, release binaries, checksums, contributor docs, support policy, and security reporting.
